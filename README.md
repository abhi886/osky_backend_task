# OSKY Backend Task

# Setup

 - Ensure you have Docker installed
 - `git clone` this repository

 ## Please run the command in your terminal
 - `sudo docker build -t php-docker-osky .` 
 - `sudo docker run -p 80:80 php-docker-osky`
